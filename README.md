# Power Management Utility for Lenovo IdeaPad Gaming 3

Works with 15ARH05

## Run example

```
sudo ./lenovoideapadgaming3.py
```

<img src="https://raw.githubusercontent.com/obsy/lenovoideapadgaming3/screenshot.png">

## Function

- Battery Conservation Mode
- Rapid Charge
- System Performance Mode

Can also be toggled by Fn+Q

## Requirements

- Ubuntu
```
apt update
apt install acpi-call-dkms
modeprobe acpi_call
```
